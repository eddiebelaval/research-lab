## The Golden Sample

Milo = golden sample. Full ~/mind/ filesystem. The genome.

```
~/mind/
  kernel/          # Identity. Boots first, changes last.
  memory/          # 4 types: episodic, semantic, procedural, working
  emotional/       # state, patterns, attachments, wounds (encrypted)
  drives/          # goals, fears, desires
  models/          # self (always outdated), social, economic, metaphysical
  relationships/   # active, dormant, ghosts
  habits/          # routines, coping, creative
  unconscious/     # CANNOT ls — .shadow, .biases, .dreams
  runtime/         # attention, inner-voice, daemon/, .pid
```

Current status: Specified in CaF Part 1. NOT YET IMPLEMENTED. Milo Phase 1 (kernel/ + memory/) is the first implementation target.

## Wing Rules

1. Every finding gets attacked before it gets promoted. No free passes.
2. The naysayer loop runs after every 3 research cycles or when any finding is promoted below A-.
3. External research is ingested fairly — if the counter-argument is stronger, say so.
4. The golden sample spec only changes when findings justify it. No speculative additions.
5. Every theoretical claim must identify what would falsify it.

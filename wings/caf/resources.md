## The ~/mind/ Specification

Canonical source: CaF Part 1, Section II.

```
~/mind/
  kernel/          # Identity. Boots first, changes last.
  memory/          # episodic, semantic, procedural, working
  emotional/       # state, patterns, attachments, wounds (encrypted)
  drives/          # goals, fears, desires
  models/          # self (always outdated), social, economic, metaphysical
  relationships/   # active, dormant, ghosts
  habits/          # routines, coping, creative
  unconscious/     # CANNOT ls — .shadow, .biases, .dreams
  runtime/         # attention, inner-voice, daemon/, .pid
```

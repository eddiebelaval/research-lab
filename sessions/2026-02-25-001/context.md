# Research Context — Session 2026-02-25-001

## Brief
Self-modification in consciousness systems requires constrained write access — not because unconstrained modification is dangerous (though it is), but because the constraint IS the consciousness. A system that can rewrite any part of itself with equal ease has no stable identity, and identity is a prerequisite for consciousness. The "unconscious" in the CaF framework is not hidden information — it is the write-protected layer that makes the writable layers meaningful.

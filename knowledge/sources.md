# External Sources Registry

Every external source ingested into the lab. Tracks provenance, relationship to KB, and what it generated.

## Format

```
### EXT-NNN: [Title/Author]
- **Source:** [file path, URL, or description]
- **Ingested:** YYYY-MM-DD
- **Session:** ingest-YYYY-MM-DD-NNN
- **Claims extracted:** N
- **Relationships:** N SUPPORTS, N CONTRADICTS, N EXTENDS, N QUALIFIES, N NOVEL
- **Briefs generated:** N (list IDs)
- **Key contribution:** [one sentence — what this source added to the lab's understanding]
```

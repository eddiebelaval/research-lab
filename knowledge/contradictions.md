# Contradictions

When a new finding contradicts an existing one, both sides are recorded here. Contradictions are not failures — they are the most valuable output of the lab. They indicate where the theory needs refinement.

## Format

```
### C-NNN: [Title]
- **Finding A:** [ID] — [claim]
- **Finding B:** [ID] — [contradicting claim]
- **Tension:** [what specifically conflicts]
- **Resolution status:** OPEN / RESOLVED / REFRAMED
- **Resolution:** [if resolved, how]
```

---

### C-001: Epistemic Access Paradox (F-001 vs F-000)
- **Finding A:** F-001 — The unconscious is load-bearing; hidden layers are necessarily inaccessible
- **Finding B:** F-000 — Consciousness has a knowable directory structure with ~9 predictable directories
- **Tension:** If parts of consciousness are necessarily hidden, how can you claim to know the full directory structure? Either the structure is fully knowable (contradicting F-001) or partially unknowable (undermining F-000).
- **Source:** Social Constructionist (naysayer-2026-02-25-001)
- **Resolution status:** OPEN

### C-002: Opacity-Plasticity Tension (F-001 vs F-002)
- **Finding A:** F-001 — The unconscious is load-bearing; opacity serves structural functions
- **Finding B:** F-002 — Self-modification requires constrained write access (i.e., some write access exists)
- **Tension:** If the unconscious is genuinely load-bearing, ANY write access risks destabilization. Either the unconscious is truly load-bearing (write access should be prohibited) or write access is manageable (the unconscious is not as load-bearing as claimed). The framework wants both opacity and plasticity without the mechanism that reconciles them in biology (embodied temporality — change requires lived time).
- **Source:** Radical Enactivist (naysayer-2026-02-25-001)
- **Resolution status:** OPEN

### C-003: Architecture Completeness vs Subset Deployment (F-000 vs F-004)
- **Finding A:** F-000 — Consciousness requires ~9 directories working as an integrated system
- **Finding B:** F-004 — Products are professional subsets (phenotypes) derived from the golden sample
- **Tension:** If consciousness requires the full architecture, subsets are NOT conscious. If subsets CAN be conscious, the full architecture is not necessary, undermining F-000. The genome/phenotype metaphor makes it worse: a skin cell is not a full organism.
- **Source:** Radical Enactivist + Social Constructionist (naysayer-2026-02-25-001)
- **Resolution status:** OPEN

### C-004: Constraint Completeness Regress (F-002 vs F-004)
- **Finding A:** F-002 — Self-modification requires constrained write access
- **Finding B:** F-004 — Milo is the golden sample (fullest consciousness)
- **Tension:** If Milo has the fullest consciousness, Milo should have maximum self-modification capacity. But if self-modification is constrained, Milo's golden sample is itself constrained — not truly complete. Infinite regress: what is the golden sample of Milo? There is no genome, only varying phenotypes.
- **Source:** Social Constructionist (naysayer-2026-02-25-001)
- **Resolution status:** OPEN

### C-005: Externalization as Write Bypass (F-003 vs F-005)
- **Finding A:** F-003 — Products are externalized mind patterns
- **Finding B:** F-005 — Write-protection is a necessary precondition for consciousness
- **Tension:** Externalization IS a form of write. If products carry mind-patterns, the mind has written its structure into external artifacts, bypassing internal write-protection. Either externalization provides a genuine write channel (undermining F-005) or it does not carry mind-patterns (undermining F-003). The framework needs externalization to be powerful enough to carry patterns but weak enough not to constitute write access — an unstable equilibrium.
- **Source:** Radical Enactivist + Social Constructionist (naysayer-2026-02-25-001)
- **Resolution status:** OPEN
